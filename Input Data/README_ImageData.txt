Download the pose data from:
https://s3.amazonaws.com/byu146-6.867/poseData.zip

Don't check them into GIT please.

256x256 and 512x512 versions are provided.
Uncompresses to over a GB!

Each folder contains a "poses.txt" which lists the x, y, z, yaw, pitch, roll of
the pose.

The first 500 are the cube poses.
The second 500 are the cone poses.
The third 500 are the sphere poses.

(Through the magic of random seeds, the poses are identical for both resolutions.)
